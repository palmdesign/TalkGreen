Branded Guest Captive Portal<\/span><\/p>\n

<\/span><\/p>\n

\n

<\/h2>\n

Now you can easily\u00a0create a fully customized, secure, responsive and user-friendly captive portal that welcomes each guest every time. Waver SmartOne comes with a powerful and easy to use built-in\u00a0editor, which lets you create your own fully-branded guest captive portal. Analytical Statistics & Graphs<\/h2>\n

 <\/p>\n

\u00a0Check the history of guest logins, connected devices minimum, average, peak hours etc. SmartOne gives you statistics for guest user data like first and last login time, emails, mobile numbers, ratings with comments from the feedback system and much more. All data are displayed in lists or\u00a0time-type based graphical charts for easy evaluation.<\/span><\/p>\n

DUAL BAND HIGH SPEED RADIOS<\/span><\/p>\n

<\/span><\/p>\n

<\/span>EASY NETWORK CONFIGURATION<\/span><\/span><\/p>\n

\n
